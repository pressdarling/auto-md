## Requirements

- Python
- Git

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Web App

```bash
python wsgi.py
```

## Docker

```bash
docker compose up --build
```

The web interface will be available at: http://localhost:8000
